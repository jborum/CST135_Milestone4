Updated Transaction class to generate final purchase - Updated driver class to incorporate the transaction class and user interface 

User must run the "Driver" class to execute the program.

The driver class creates a Dispenser object, loads the dispenser, launches the displayProducts() method and launches the user interface.

The Transaction class creates a transaction object to store the objects that were purchased.

The user shall click on the button to open the category of choice, drinks, candy, chips or gum.

Once the category button is selected the individual items are loaded for the user to select. The user can select the product and a message will display showing the user they are going to purchase the item. The user can close the window by clicking the close button.

When the user is ready to checkout they can select the checkout button on the main menu. The checkout button will display the total purchase for the transaction.
